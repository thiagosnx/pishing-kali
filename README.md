## Tecnologias
- Kali Linux
- Social-Engineer Toolkit

## Passo a passo

- No terminal do Kali Linux, começamos com os seguintes comandos

- Para ter privilegios root:
```
sudo su
```
- Iniciando o seetolkit:
